- 전체 페이지 수 => 올림ceil(전체 아이템 수 / 페이지당 아이템 수) = 6
- 전체 섹션 수(ex : 1~5, 6~10...) => ceil(전체 페이지 수 / 섹션당 페이지 수) = 2
- 시작 인덱스 = 페이지당 아이템 수 * (현재 페이지 - 1)
- 종료 인덱스 = 시작인덱스 + (페이지당 아이템 수 - 1)
만약 종료 인덱스가 전체 길이 -1보다 크다면 "종료 인덱스" = "전체 길이 -1"
